T4 HAND REVIEW

HJ A♥J♥ の 3bet pot turn raise レビュー

主論点は、SB の 3bet pot で J♣ 6♠ 5♥ 2♥ に対する 2bar range をどう捉え、IP の A♥J♥ を call bucket と raise bucket のどちらに置くかです。結論は、preflop の continue と flop の call はかなり自然で、いちばん見直すなら turn の raise です。A♥J♥ は強い continue ですが、IP では raise より call の方がレンジに対して収まりやすいです。

Hero: HJ A♥J♥
Villain: SB A♠A♣
Board: J♣ 6♠ 5♥ / 2♥ / 5♣
  • GTO基準の総評: preflop call、flop call は良く、主なズレは turn raise です。raise 後に 3bet を返されてからの call 自体はそこまで不自然ではありません。
  • SB の turn half-pot range には AA-KK-QQ, set, 一部 heart semibluff がかなり残ります。A♥J♥ はそのレンジに対して強い continue ですが、IP なので call でほぼ全 equity を実現できます。
  • A♥ を持っていることで、相手の自然な heart semibluff (A♥K♥, A♥Q♥) を block しています。raise は相手の bluff / semibluff を減らした状態で、強い value にぶつかりやすい構造です。

PreflopSB の 3bet range は blind から HJ open に対してかなり強く、QQ+, AK/AQ, suited broadway が厚いです。A♥J♥ は IP の continue 候補として十分強く、4bet より call が自然です。良いここは問題ありません。
Flop J♣ 6♠ 5♥SB の small c-bet には overpair, AK/AQ, backdoor 付き broadway が広く含まれます。A♥J♥ は top pair top kicker で、相手レンジに対する強い call bucket です。良いraise を急がず call に置けているのはかなり自然です。
Turn 2♥SB の 2bar range は依然として overpair と set が濃く、heart semibluff も一部残ります。A♥J♥ は pair + nut flush draw で強い continue ですが、IP なので call の実現値が非常に高いです。ややオーバープレイraise の発想は分かりますが、レンジに対しては call の方がきれいです。
River

持ち帰り

  • IP の top pair + nut flush draw は、まず call で十分に equity を取り切れるか を先に見ると無理な raise が減ります。
  • 強い combo ほど raise したくなりますが、相手の bluff を消して value に寄せていないか を同時に確認すると精度が上がります。
  • turn で pot を膨らませる判断と、膨らんだあとに call する判断は別物として分けて振り返ると整理しやすいです。

exploit 補足

  • 3bet pot の turn で raise にさらに返してくる line は pool でかなり value 寄りになりやすいので、最初の raise を絞るだけで大きく事故が減ります。
  • それでも A♥J♥ のような pair + nut flush draw は continue 候補に残りやすく、同じ感覚を弱い draw にまで広げないのが大事です。

Izacchi SB A♠K♥ の 3way preflop all-in レビュー

主論点は、BTN open に対する SB 3bet、BB cold 4bet、BTN 5bet jam、SB call、BB fold という 3way preflop all-in を、参加した 3 人それぞれの立場でどう評価するかです。結論は、SB A♠K♥ の 3bet 自体は自然ですが、BTN J♦J♣ の jam は前のめり、SB AKo の call も offsuit ではかなり際どく、いちばん見直したいのは BB Q♥Q♣ の small cold 4bet とその後の fold です。showdown で SB が勝っていても、preflop の良し悪しは別で切り分けたい hand です。

  • BTN Riku0125 の J♦J♣ open は自然ですが、SB 3bet + BB cold 4bet に挟まれた状態での 100BB jam は強く進めすぎです。SB の continue と BB の cold 4bet call にぶつかりやすく、default は fold 寄りで見たいです。
  • SB Izacchi の A♠K♥ 3bet は素直です。ただし BB 20BB cold 4bet -> BTN jam を見てからの AKo call はかなり薄く、しかも BB が後ろに残るので、理論寄りには AKs や QQ+ より厳しく扱いたい帯です。
  • BB nu0un の Q♥Q♣ cold 4bet 自体は value です。ただ 20BB はやや小さく、結果として BTN に jam しやすい価格を与え、SB にも continue を残しやすくしています。
  • その一方で、BB は jam + call を見たあと closing action で 80BB を足して 300BB pot を取りにいけるので、必要 equity はおよそ 26.7% です。QQ は default で call が十分あり、fold は read 強めの exploit 寄りです。
  • この hand は「勝った line が正しい」と見ない方が学びやすい例で、3人それぞれの node で必要な基準がかなり違います。

持ち帰り

  • 3way preflop all-in は、誰が closing か と 何%必要か を席ごとに分けて見ると整理しやすいです。
  • JJ、AKo、QQ はどれも強い hand ですが、cold 4bet -> jam -> call まで伸びると続ける基準はかなり変わります。
  • small cold 4bet は一見安く見えても、後続に jam と overcall の余地を残しやすいので、強い value ほど size で tree を締めた方が安定します。

exploit 補足

  • BB の cold 4bet が small で、その後に overfold まで多い pool なら、BTN / SB が exploit で押し返す余地はあります。
  • 逆に匿名 pool の cold 4bet -> jam call が AA/KK 寄りにかなり締まるなら、SB の AKo fold はかなり実戦的です。
  • BB は実戦で multiway all-in を嫌って fold しすぎる人もいますが、基準線としては 26.7% という price を忘れない方がぶれにくいです。

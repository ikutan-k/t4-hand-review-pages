ストリート別レビュー

ストリートその時点の見え方判断の焦点評価推奨
PreflopBTN open は広く、BB は suited Ax や broadway を多く defend できます。AsTs は十分 defend できる hand で、ポストフロップも扱いやすいです。良いCall で問題ありません。

Turn AcBTN の check-back range には Ax、中位の showdown hand、slowplay がまだ十分残ります。Hero の hand は強化されましたが、BB range 全体が優位になるわけではありません。やや微妙GTO寄りには check がかなり有力です。打つなら小さく広くより、もう少し極化した設計の方が筋が通ります。
River Qsturn call 後の BTN には多くの Ax、一部 8x、slowplay、spade が残ります。AsTs は paired board でもかなり上位の value hand です。良い100-125%pot の大サイズは十分自然です。

持ち帰り

  • turn では 自分の hand が強くなったから打つ だけでなく、このカードで誰の range が強くなるか まで見ると判断が安定します。
  • Ac は Hero には嬉しい一方で、flop check-back 後の BTN range もかなり助けるカードです。
  • river は逆に、nut flush まで改善したならしっかり value を取りにいく発想が良く、今回の大サイズはかなり擁護できます。

exploit 補足

  • pool が river で raise 不足かつ Ax を降ろしにくいなら、今回の 125%pot は実戦でもかなり強いです。
  • 一方、paired flush board で one pair をしっかり降ろせる相手には、小さめ size の方が広く call を拾える場面もあります。

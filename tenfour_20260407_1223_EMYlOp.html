T4 HAND REVIEW

BB Q♣6♣ の flop float と river overbet レビュー

主論点は、A♥ K♣ 4♥ で Q♣6♣ を check-call した flop の float がどこまで支えられるかです。当時の思考メモはないので、「A-high board でも backdoor が少しあるし 1回は見たい」と考えた仮定で見ます。結論は、preflop defend と river の overbet lead は自然で、いちばん見直すべきなのは flop の continue です。

Hero: BB Q♣6♣
Villain: BTN J♥T♥
Board: A♥ K♣ 4♥ / T♦ / J♣
  • GTO基準の総評: 一番大きいズレは flop の call で、それ以外は大筋で自然です。
  • A♥ K♣ 4♥ に対する Q♣6♣ は、pair も straight draw もなく、half-pot を OOP で受けるにはかなり弱いです。default は fold です。
  • turn check-check で BTN range が capped になったあと、river で straight を作って大きく取りにいく line は良いです。

flop の Q♣6♣ callQ-high でも overcard があり、backdoor club もあるので 1回くらい見たくなるこの board は BTN の A/K 優位がかなり強く、Q♣6♣ は pair も straight draw もありません。3.25BB を払っても clean な turn が少なく、OOP では実現値も落ちやすいです。A/K 高ボードの half-pot に対しては、pair, heart draw, より強い backdoor を優先し、Q-high の pure float は fold baseline にする
結果の受け取り方river で straight まで行ったので、flop call も正しかった気がする今回は turn / river がかなり都合よく落ちただけで、flop 時点の continue 根拠は薄いままです。結果で前段の判断を美化すると、同じ overfloat が増えやすいです。勝ったかどうか ではなく、その時点で continue の根拠が足りていたか で振り返る

ストリート別レビュー

ストリートその時点の見え方判断の軸評価コメント
PreflopBTN open に対して BB はかなり広く defend できます。Q♣6♣ は 3BB open に対する BB defend として自然です。良いここは問題ありません。
Flop A♥ K♣ 4♥BTN は Ax, Kx, heart draw, broadway を広く持ち、BB 側はかなり守備的になります。Hero は Q-high で、pair も straight draw もなく、backdoor club しかありません。明確に広すぎる3.25BB に対して続ける根拠が足りません。ここは素直に fold で大丈夫です。
Turn T♦flop を残した結果、Hero は broadway への gutshot を拾います。BTN の check-back で IP range はやや capped になります。Hero にはまだ showdown value がなく、river を見にいく形で十分です。自然この street の問題は小さく、根本修正点は flop を残しすぎないことです。
River J♣turn check-check 後の BTN には Ax, Kx, JT, missed heart draw, 一部 Qx が残ります。Hero は broadway でかなり強く、capped な range に対して大きく取りにいけます。良い19.5BB の overbet lead はかなり良いです。two pair や curiosity call をしっかり取りきれます。

持ち帰り

  • A/K 高ボードで OOP から続けるときは、pair or draw が本当にあるか を先に確認すると overfloat が減ります。
  • river で強く当たったときは、turn check-check で capped になった相手 range に対して大きく打つ発想はそのまま持っていて大丈夫です。
  • 勝ったハンドほど、どこから EV が出ていたか と どこは運で救われたか を分けて見ると精度が上がります。

exploit 補足

  • この workspace の暫定傾向どおり、turn check-check 後は IP range が capped になりやすく、river lead は実戦でも機能しやすいです。
  • ただし exploit で flop defense を広げるなら、heart を含む backdoor や blocker がもう少し強い combo から始める方が安全です。

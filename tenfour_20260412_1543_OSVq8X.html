T4 HAND REVIEW

BB K♥J♠ の monotone flop / A-turn overbet 対応レビュー

主論点は、BB defend 後の K♦ Q♦ 2♦ で K♥J♠ をどこまで continue し、A♠ turn の 150%pot overbet に対してどの bluff-catch bucket まで残すかです。結論は、preflop defend と flop call は大筋で自然で、turn fold はかなり良い整理です。いちばん大事なのは、monotone flop の小さい c-bet を弱さと決めつけず、逆に pot が大きくなるほど no-diamond の one-pair を早めに手放すことです。

Hero: BB K♥J♠
Villain: BTN 9♦8♦
Board: K♦ Q♦ 2♦ / A♠
  • GTO基準の総評: preflop defend、flop call、turn fold はかなり自然です。特に turn fold は無理に守らなくてよい spot です。
  • K♦ Q♦ 2♦ の monotone flop では、BTN は小さいサイズを高頻度で使いやすく、1.83BB は弱さのサインではありません。K♥J♠ は top pair ですが diamond を持たず、強い continue ではなく中間の check-call 寄りです。
  • A♠ turn で pot が大きくなり、しかも 150%pot の overbet まで出ると、BTN range は flush、JT、高い diamond blocker を使った bluff にかなり寄ります。K♥J♠ は gutshot を得ても、no-diamond の one-pair としては降りて問題ありません。

持ち帰り

  • monotone flop の小さい c-bet は、小さい = 弱い ではありません。まずは range bet に近いものとして受け取ると整理しやすいです。
  • diamond を持たない top pair は、flop で1回守れても、pot が大きくなった時点で一気に価値が落ちます。
  • まだ top pair かどうかより、flush blocker を持つか、相手の polar size に何を残せるか を先に見ると、turn の判断がかなり安定します。

exploit 補足

  • 実戦では monotone board の大きい turn aggression は GTO以上に value 寄りになりやすいので、K♥J♠ のような no-diamond one-pair はさらに降りやすくて大丈夫です。
  • 相手が blocker を使って overbluff するタイプでも、先に守るべきなのは high diamond や two pair 以上で、diamond なしの Kx から無理に広げる必要はありません。

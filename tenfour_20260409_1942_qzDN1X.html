T4 HAND REVIEW

UTG K♦Q♦ の 3bet pot delayed line 対応レビュー

主論点は、HJ の 3bet pot で A♠ 6♥ 2♣ の flop check-back にどんなレンジが残り、Q♥ turn の小さい delayed c-bet と 7♦ river の pot bet に対して K♦Q♦ をどの bluff-catch bucket に置くかです。結論は、preflop call、turn call、river fold は大筋で自然です。いちばん大事なのは、flop の check-check を弱さと決めつけず、river の大きい size では value density を見直すことです。

Hero: UTG K♦Q♦
Villain: HJ A♦Q♣
Board: A♠ 6♥ 2♣ / Q♥ / 7♦
  • GTO基準の総評: preflop call、turn call、river fold はかなり自然です。特に river fold はレンジ主導で見ると良い整理です。
  • HJ の flop check-back range には、AK/AQ の一部、KK-JJ-TT、少数の trap が十分残ります。turn Q♥ で K♦Q♦ はそのレンジに対する自然な call bucket へ上がります。
  • ただし river 7♦ の pot bet まで進むと、HJ range は AQ, AA, QQ, 一部 AK にかなり寄ります。K♦Q♦ は KJ/QJ 系の bluff を block するので、上側の bluff-catcher ではあるものの fold 寄りに寄せやすい combo です。

持ち帰り

  • 3bet pot の A-high flop で IP が check-back しても、相手 range はまだ十分強いです。空気になった と見ないことが大事です。
  • turn で pair を作ったときは、その hand が 相手の delayed range に対してどの call bucket に入るか を先に見ると整理しやすいです。
  • river の pot bet は、自分の hand が上位 bluff-catcherか だけでなく、何の bluff を block しているか まで見ると精度が上がります。

exploit 補足

  • この workspace の暫定傾向どおり、大きい river aggression は value 寄りになりやすいので、KQ のような one-pair bluff-catcher は少し降り寄りで十分戦えます。
  • HJ が delayed line から river で pot を打つタイプでも、まずは AQ+/slowplay の残存を強めに見る方が安定します。

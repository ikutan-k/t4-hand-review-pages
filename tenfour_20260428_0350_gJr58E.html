T4 HAND REVIEW

UTG 5♦5♣ の multiway set value / river overbet レビュー

GTO基準で大きくズレているのは Hero 側ではなく、むしろ Villain 側の A♥9♥ での call down の広さです。Hero の 5♦5♣ は preflop open から、flop set、turn 大きめ value まではかなり自然です。見直すなら river J♣ で 79.68BB を 49.97BB に overbet all-in した点で、これは「set だから常に最大を打つ」ではなく、「相手が Ax をかなり降りない pool なら強い exploit」という位置づけです。

Hero: UTG 5♦5♣
Villain: CO A♥9♥
Board: A♠ 5♥ 4♦ / 7♦ / J♣
  • UTG 5♦5♣ の open は自然です。2BB open で後ろから複数 call されても、set value があり、preflop の大きな問題ではありません。
  • flop A♠ 5♥ 4♦ は 5way ですが、Hero は bottom set でかなり強い value hand です。3.33BB into 10BB の small c-bet は、Ax、pair、straight draw を広く残せるので良いです。
  • turn 7♦ は diamond draw と一部 straight が増えるため、Hero は protection も兼ねて大きく打てます。14.99BB into 19.99BB は良い value size です。
  • river J♣ は flush は完成せず、多くの draw は外れています。ただし CO の flop call / turn call range には、強い Ax、two pair、set、まれな straight も残るため、bottom set の巨大 overbet は標準より exploit 寄りです。

持ち帰り

  • multiway で set を作ったとき、flop/turn は「小さく入れて広く残す」「turn で大きく value/protection」の流れが扱いやすいです。
  • river の大きい bet は、こちらの hand の強さだけでなく、相手が worse hand で call する密度を見て決めます。bottom set は強いですが、常に最大 bet ではありません。
  • 今回の pool では Villain が A9s top pair で overbet all-in まで call しているため、Hero の river shove は exploit としてかなり良い結果になっています。

exploit 補足

  • CO の A♥9♥ は UTG open に対する preflop call から river call まで、全体にかなり広いです。このタイプが多いなら、Hero は強い value hand で size を大きくしてよいです。
  • 逆に、river の大きい bet に Ax をきちんと fold できる相手には、bottom set の overbet all-in は call されたときの相手 range が強くなりすぎます。その場合は 1/2 pot から pot 前後の value bet に寄せる方が安定します。

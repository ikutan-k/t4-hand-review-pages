T4 HAND REVIEW

BB A♦T♦ の river thin value / raise fold レビュー

GTO基準で大きな問題はありません。BTN 2BB open に対する BB A♦T♦ defend、flop T♣ 8♥ 2♠ の check-call、turn 4♠ の check は自然です。主な改善点は river 4♥ で 5.63BB into 7.5BB まで大きく thin value を打った点です。A♦T♦ はかなり強い top pair ですが、river で board が pair したあとに大きく打つと、worse Tx/8x から取りにくくなり、raise を受けたときに難しくなります。実戦の大きい raise に対する fold は良い判断です。

Hero: BB A♦T♦
Villain: BTN 5♦4♦
Board: T♣ 8♥ 2♠ / 4♠ / 4♥
  • BB A♦T♦ は BTN min open に対して call で問題ありません。3bet mix に回せる hand ですが、call は標準的です。
  • flop T♣ 8♥ 2♠ は Hero が top pair top kicker を持っており、BTN の 1/3 pot c-bet に check-call するのは自然です。
  • turn 4♠ の check-check 後、BTN には air も残りますが、4x、slowplay、Tx、8x もまだ残ります。turn check back だけで強い hand を消し切らないのが大事です。
  • river 4♥ の bet は value として成立しますが、サイズは小さめから中サイズの方が扱いやすいです。大きい raise への fold は、実戦の river raise が value 過多になりやすいためかなり堅実です。

持ち帰り

  • BTN が turn を check back しても、river raise の value range まで消えるわけではありません。特に BTN は広いので 4x が自然に残ります。
  • top pair top kicker の river value は成立しますが、paired board では大きく打つほど thin になります。worse Tx/8x から取りたいなら、小さめから中サイズを優先します。
  • 今回の fold は良いです。Hero は 4x blocker を持っておらず、相手の大きい raise に対して A♦T♦ は bluff-catcher です。

exploit 補足

  • T4 の実戦では、river の大きい raise は理論より value に寄りやすいです。この傾向を前提にするなら、今回の bet-fold は十分堅実です。
  • ただし相手が river raise bluff を明確に多用する場合だけ、top pair 上位での bluff-catch を検討します。その場合でも、4x blocker を持つ hand や、より上位の value hand を優先したいです。

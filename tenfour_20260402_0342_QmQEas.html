BTN 3bet pot AQo レビュー

主論点は、3bet pot の A-high board で AQ をどこまで押し切るかです。当時の思考メモはないので、「top pair top kicker で pot control したくなった」という仮定で見ます。結論は、preflop 3bet、flop c-bet、river fold は自然で、いちばん見直したいのは turn の check でした。

持ち帰り

  • この hand は river の hero call 可否より、turn で value / protection を取りにいけたかを見る方が学びになります。
  • AQ のような強い one pair は、強いから check ではなく、まだ worse が続くか と draw に無料カードを渡したいか で判断すると整理しやすいです。
  • overbet は QQ+ や極化 hand 向きで、AQ は広くコールされる medium size の方が噛み合いやすいです。

exploit 補足

  • pool が turn で overfold するなら大きい size 自体は武器になりますが、AQ 本体は fold されて困る hand にも勝っています。
  • 実戦でも AQ は medium size の方が AJ/AT や draw から取りやすく、安定して稼ぎやすいです。

T4 HAND REVIEW

BB J♥8♥ の flop float / turn no-diamond check-raise bluff レビュー

GTO基準で明確にズレやすいのは、flop Q♥ 7♦ 2♦ の call と、turn A♦ で J♥8♥ を check-raise bluff に回した点です。Preflop の BB defend は問題ありません。ただし postflop は、no pair / no diamond / 低い backdoor だけの hand で OOP から粘り、さらに flush 完成 turn で大きく check-raise しているため、理論上は bluff 候補としてかなり下側です。結果は BTN が A♣9♥ を fold して成功していますが、これは exploit 寄りの成功として扱うのが安全です。

Hero: BB J♥8♥
Villain: BTN A♣9♥
Board: Q♥ 7♦ 2♦ / A♦
  • BB J♥8♥ は BTN 2.5BB open に対して defend 可能な suited hand です。preflop call 自体は自然です。
  • flop Q♥ 7♦ 2♦ は BTN 側に強い Qx、overpair、diamond draw が多く、BB は広く守るとはいえ OOP で実現率が低いです。J♥8♥ は no pair / no diamond で、call するにはかなり薄い hand です。
  • turn A♦ は BTN の Ax に当たり、さらに diamond flush も完成します。BB 側も flush は持てますが、J♥8♥ 自体は equity も blocker も足りません。
  • turn check-raise bluff を作るなら、single diamond、pair + diamond、straight draw + diamond、または相手の強い continue をブロックする hand を優先します。今回の J♥8♥ はその条件から外れています。

持ち帰り

  • BB vs BTN の preflop defend が広いことと、postflop で何でも粘れることは別です。OOP では、安い flop bet でも turn 以降に戦える材料が必要です。
  • flush 完成 turn で bluff raise するなら、相手の強い continue をブロックする diamond 持ち、または call / raise 後にも equity が残る hand を優先します。
  • 今回は Villain が A♣9♥ top pair no diamond を fold して成功しましたが、J♥8♥ のような no-blocker hand で再現しにいくと過剰 bluff になりやすいです。

exploit 補足

  • BTN が flop small c-bet から turn で A♦ を広く bet し、check-raise に top pair no diamond をすぐ fold する傾向があるなら、このライン自体は exploit として利益が出ます。
  • それでも bluff combo の選び方は改善できます。同じ exploit を狙う場合でも、J♥8♥ より diamond blocker を持つ hand を優先した方が、call されたときの equity と相手の strong range への圧が残ります。

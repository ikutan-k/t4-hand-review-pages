T4 HAND REVIEW

BTN K♣J♣ の CO 3bet pot river thin value レビュー

主論点は、CO 2.5BB open に対して BTN K♣J♣ で 7.5BB 3bet し、flop K♠ 6♦ 5♥ で small c-bet、turn T♥ check back、river 4♦ で small value bet した判断です。結論は、postflop はかなり良いです。見直すなら preflop で、BTN の KJs は CO open に対して defend は明確ですが、GTO基準では call に残しやすい hand です。3bet は相手が overfold する、または call 後に postflop で利益を出せるなら使える、という位置づけです。

Hero: BTN K♣J♣
Villain: CO A♦T♦
Board: K♠ 6♦ 5♥ / T♥ / 4♦
  • GTO基準の総評: K♣J♣ は BTN vs CO open で続行する hand ですが、標準は call 寄りです。今回の 3bet は攻撃的寄せで、明確な大ミスではありません。
  • flop K♠ 6♦ 5♥ は BTN 3bet 側に AA/AK/KQ と overpair があり、レンジ優位を取りやすい board です。KJ top pair は small c-bet で value/protection を取れます。
  • turn T♥ は CO の TT/KT/ATs/QJs などにも関係し、Hero の KJ は3 streets 大きく取り切るほど強くありません。check back は自然です。
  • river 4♦ で CO が3回目の check をしたあと、Hero の KJ は小さく value bet できます。Tx、弱い Kx、一部 pocket から call をもらう狙いで、9.17BB は良い thin value size です。

持ち帰り

  • BTN KJs は CO open に対して強い defend hand ですが、標準は call に残しやすいです。3bet は相手依存の攻撃的寄せとして使います。
  • 3bet pot の dry K-high flop で top pair を作ったら、small c-bet は自然です。相手の広い continue を残しながら value/protection を取れます。
  • turn で中間化した top pairは、check back して river の小さめ value に回すと扱いやすいです。今回の river size は、勝っている相手の call range をよく意識できています。

exploit 補足

  • 実戦の CO は A♦T♦ で flop call、turn check、river T pair で call しています。この pool が second pair を river small bet に call してくれるなら、今回の thin value はかなり利益的です。
  • 相手が river で Tx をすぐ fold するなら、river は check back 寄りになります。逆に今回のように call が広いなら、turn check back から river small value を増やしてよいです。

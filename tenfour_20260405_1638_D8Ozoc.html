SB vs BB AJo レビュー

主論点は、top pair good kicker を 3street 取り切る hand と見るか、turn で一度抑えて river で thin value を取り直す hand と見るかです。当時の思考メモはないので、「turn でもまだ打ちたくなる」あるいは「river 75%pot は少し厚いかも」という仮定で見ます。結論は、line 全体は大筋で良く、turn check と river value の組み合わせはかなり筋が通っています。

Hero: ikutaN SB AdJh
Villain: BB
Board: Jc 5d 2s / 2d / 9c

  • 結論として、preflop open、flop c-bet、turn check、river value はかなりまとまっています。

持ち帰り

  • AJo のような top pair good kicker は、turn で無理に 3発打ち切らず、river の thin value に回す設計がよくあります。
  • pair turn は まだ強いから打つ ではなく、相手の弱い range を残した方が得か で見ると判断しやすいです。
  • river は turn の受け身を見たあとなら、やや厚めの value でも十分成立します。

exploit 補足

  • pool が river で A-high や弱い Jx を降ろしにくいなら、この 75%pot はかなり取りやすいです。
  • 相手が大きめ size にかなり素直に降りるなら、少し小さくして広く call を拾う調整もあります。

T4 HAND REVIEW

SB iso raise vs UTG limp A5s レビュー

主論点は、A5s を limp に対する iso raise として使えたうえで、flop の A-high を自動 c-bet しない discipline を持てているかです。当時の思考メモはないので、「iso raiser だから打たないといけないか迷った」という仮定で見ます。結論は、preflop から river まで大きなズレはなく、むしろ過剰に bluff を作らなかった点が良い review です。

Hero: SB Ac5c
Villain: UTG
Board: Jh 4d 2c / Ts / 6h
  • GTO基準の総評: A5s の iso raise とその後の受け身ラインは大筋で自然で、無理に bluff を足さなかったのも良い判断です。
  • 当時の思考メモはないので、iso raiser だから flop を打たないといけないか迷う という仮定でレビューします。
  • 結論として、A5s の iso raise はかなり自然で、flop / turn / river の受け身ラインも十分許容です。

持ち帰り

  • この hand は、A5s を SB から iso できている時点でかなり良いです。
  • flop は c-bet してもよいですが、iso raiser だから必ず打つ にしないことが大事です。
  • 改善しないまま無理に押し込まないのも、実戦ではかなり大事な discipline です。

exploit 補足

  • limp-call する相手が flop を過剰に降ろすなら、Jh 4d 2c での small c-bet はかなり通ります。
  • pocket や weak pair で粘る相手なら、今回のように check 寄りで進めるのも実戦的です。

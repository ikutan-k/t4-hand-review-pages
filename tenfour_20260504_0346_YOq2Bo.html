T4 HAND REVIEW

BB A♠3♠ の paired river nut flush 3bet レビュー

GTO基準で、CO 2BB open に対する BB A♠3♠ の call は自然です。flop J♠ 7♠ 2♠ で nut flush を完成し、CO の 1/4 pot c-bet に check-call した判断も良いです。turn 3♣ の check-check後、river 2♥ で小さく lead したところまではかなり自然です。見直したいのは、CO の raise 5.07BB に対して 10.14BB へ 3bet した点です。Hero は nut flush ですが、river で board が paired したため full house / quads には負けます。標準は call 止め寄りで、3bet は相手が lower flush や trips を raise-call しすぎる時だけに寄せたいです。

Hero: BB A♠ 3♠
Villain: CO Q♣ 5♣
Board: J♠ 7♠ 2♠ / 3♣ / 2♥
  • GTO基準の総評: preflop、flop slowplay、river small lead までは良いです。river 3bet はやや攻めすぎです。
  • このハンドの主要論点: paired river で nut flush をどこまで value raise できるか。
  • 一番大きい改善ポイント: nut flush と nuts を分ける。board が paired したら full house が上に残ります。
  • CO の Q♣5♣ raise はかなり広い bluff です。ただし bluff 相手には 3bet しても追加 value はほぼ増えず、call で十分勝てます。

持ち帰り

  • monotone flop で nut flush を作った時、small bet への check-call は相手の弱い range を残せる良い選択です。
  • river で board が paired したら、nut flush でも full house に負けます。
  • 小さい lead に raise された時は、相手の bluff を降ろす価値 ではなく、worse value が3betに call するか を基準にする。
  • paired river の nut flush は、標準では bet-call 寄り。3bet は exploit 条件がある時だけにします。

exploit 補足

  • CO が Q♣5♣ のような no spade / no pair で river small lead に raise するなら、small lead 自体はかなり有効です。
  • ただしこの相手が bluff raise 多めなら、nut flush は3betで降ろすより call で十分です。下の flush や trips でも raise-call してくる相手だと確認できてから、3bet value を増やします。
